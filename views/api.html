Overview
The weather data API consists of various routers built on the Express/Node.js framework. Data are stored as documents in MongoDB. Currently, data consist of daily high and low temperatures in Minneapolis dating to 1 January 1891.

API access requires a key. Your key is listed above if you are signed into the system. Please register an account to generate an API key.

There are several options for accessing data as outlined in the examples below.
Year Data
Returns daily data for the year specified.
{{ api.url }}/weather/api/{{ api.userdata.user.apiKey }}/years/1985
Month Data
Returns daily data for all days, for all years, for month specified.
{{ api.url }}/weather/api/{{ api.userdata.user.apiKey }}/months/10
Month/Day Data
Returns data for specified day in specified month for all years.
{{ api.url }}/weather/api/{{ api.userdata.user.apiKey }}/months/10/days/26
Year/Month Data
Returns daily data for specified month in specified year.
{{ api.url }}/weather/api/{{ api.userdata.user.apiKey }}/years/1985/months/10
Year/Month/Day Data
Returns data for specified day in specified month in specified year.
{{ api.url }}/weather/api/{{ api.userdata.user.apiKey }}/years/1985/months/10/days/26
